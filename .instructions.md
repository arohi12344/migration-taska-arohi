## Principles for This Project

1. **Preserve Business Logic** — Never alter core business logic without explicit instruction
2. **Zero Loss of Functionality** — Account for every code construct; ensure complete coverage
3. **Incremental Validation** — Migrate in logical units; validate each section before proceeding
4. **Systematic Approach** — Follow the 10-phase workflow defined in SKILL.md
5. **Quality Gates** — Implement critical checks at each phase to catch issues early
6. **Complete Traceability** — Maintain clear mapping between source and migrated code
